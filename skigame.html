SkiHD: The Open-Source HD Clone of SkiFree

SkiHD (or as the crappy name is GameMakerSkiGame (GMSG) is a Open Source light clone of Chris Pirih's classic Windows 3.1 game SkiFree. This was developed with GameMaker Studio 1.4.9999 and can be (hopefully) painlessly ported to GameMaker Studio 2.


Engine Used: GameMaker Studio
